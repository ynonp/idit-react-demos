Installation:

1. npm install -g yarn (one time per computer)

2. yarn init (once per project)

3. yarn add -d react (once per project)
               react-dom
               webpack
               webpack-cli
               babel-cli
               @babel/core
               @babel/preset-react
               html-webpack-plugin
               clean-webpack-plugin
               babel-loader

4. Config webpack

5. Write some React

6. Build with: npx webpack
